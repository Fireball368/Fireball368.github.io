HTML Table Reference

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<th> Table Header The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table data container.
<tfoot> Table Foot The set of rows defining the footer in a table.
Attribute Name Description
colspan Column Span Defines the number of columns a cell should span.
rowspan Row Span Defines the number of rows a cell should span.
headers Headers Defines a space-separated list of header cells that provide information for the cell.
scope Scope Defines the set of header cells a cell is related to.
abbr Abbreviation Defines an abbreviated version of the content in a cell.
colgroup Column Group Defines a group of columns in a table.
rowgroup Row Group Defines a group of rows in a table.
